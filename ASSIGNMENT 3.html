(1)    Introduction to the Services:

This web application includes 2 services. They are running in parallel.

·         Credit Card Validator: The service takes 16-digit credit card number, CVV, and expiration date of the credit card. The service follows Luhn’s algorithm to check whether the credit card is VALID or INVALID to use. You can check out the algorithm here: https://en.wikipedia.org/wiki/Luhn_algorithm

·         Web to String: The service takes URL of a website and returns its content as string file. This service is a remote service from ASU Service Respository. The service reference is here: http://neptune.fulton.ad.asu.edu/WSRepository/Services/Web2StringSVC/Service.svc

(2)    Description of the interface:

·         Credit Card Validator:

The service has three textboxes for credit card number, CVV, and expiration date. The button “Check Credit Card” is for validating the given card.

·         Web to String:

The service has URL textbox. User must include http:// before the website address in order to run service. The “SUBMIT” button is used to convert a website to the string content. CONTENT area shows the result of the service. User must manually clear the content area before submit a new website.
